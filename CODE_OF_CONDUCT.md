# Code of Conduct

## Our Pledge

Contributors and maintainers are expected to make participation in this project a respectful and harassment-free experience for everyone.

## Expected behavior

- Be respectful and constructive
- Assume good intent
- Welcome feedback and different viewpoints
- Focus on what is best for the project
- Use clear and professional language

## Unacceptable behavior

- Harassment or abusive language
- Personal attacks
- Trolling or deliberate disruption
- Publishing private information without permission
- Discriminatory language or behavior

## Enforcement

Project maintainers may remove, edit, or reject comments, commits, code, issues, and other contributions that violate this Code of Conduct.

## Reporting

Report unacceptable behavior through the process described in [SECURITY.md](./SECURITY.md) or the support channels in [SUPPORT.md](./SUPPORT.md).
